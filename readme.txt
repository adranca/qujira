Used to automaticaly generate ticket id and URL pointing to that ticket for a JIRA page. Simply CMD+C on any Jira page. The output saved in clipboard and follows the format [TICKET-ID](TicketURL). This can be helpful when opening a bitbucket/github ticket to add the ticket url in description. 

Chrome

https://developer.chrome.com/docs/extensions/mv2/faq/#faq-dev-01

Go to chrome://extensions/
Enable developer mode
Click "Load unpacked" 
Select the "chrome" folder
Done.

Safari

https://developer.apple.com/documentation/safariservices/safari_app_extensions/building_a_safari_app_extension

Build the xcode project in the "safari" folder
Follow the guide to install the developer extension
